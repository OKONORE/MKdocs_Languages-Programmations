    Definition

    Python is an interpreted, multi-paradigm, cross-platform programming language. It promotes structured, functional and object-oriented imperative programming. It has strong dynamic typing, automatic memory management by garbage collection and an exception handling system. source: Wikipedia1

    • Functional Oriented
    • Object Oriented
